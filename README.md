# 🌍 GeoCamaçari

🚀 Projeto de Geoprocessamento Web

## 📌 Sobre o projeto
O GeoCamaçari é uma aplicação web de geoprocessamento desenvolvida para a região de Camaçari (BA), com foco na visualização e análise de dados geográficos.

A aplicação permite visualizar um mapa interativo, identificar a localização do usuário e acompanhar dados climáticos com atualização em tempo quase real.

---

## 💻 Tecnologias utilizadas
- HTML  
- CSS  
- JavaScript  
- Leaflet (mapas interativos)  
- API de clima (Open-Meteo)

---

## 📊 Funcionalidades
- 🌍 Mapa interativo  
- 📍 Geolocalização do usuário  
- 📌 Exibição de coordenadas  
- 🌡️ Monitoramento climático automático
